# Barbearia Inteligente

Sistema de gestão para barbearias com agendamento inteligente e programa de fidelidade.

## Como subir para o Git (GitHub/GitLab)

1.  **Inicie o repositório local:**
    ```bash
    git init
    ```

2.  **Adicione os arquivos:**
    ```bash
    git add .
    ```

3.  **Faça o primeiro commit:**
    ```bash
    git commit -m "feat: versão estável com sistema de agendamento e regras de segurança"
    ```

4.  **Conecte ao seu repositório remoto:**
    ```bash
    git remote add origin <URL_DO_SEU_REPOSITORIO>
    ```

5.  **Envie o código:**
    ```bash
    git push -u origin main
    ```

### Onde usar o Token do Git?
Quando você executar o comando `git push`, o Git solicitará:
- **Username:** Seu usuário do Git.
- **Password:** **Cole o seu Token aqui**. (Nota: O cursor não se moverá e nada aparecerá enquanto você cola, por segurança. Apenas cole e aperte Enter).
